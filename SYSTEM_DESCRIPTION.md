# TAIWA AI システム概要

## 1. プロジェクト概要
**TAIWA AI** は、マネージャーとメンバーの認識ギャップを解消し、心理的安全性の高い自律的な組織文化を構築するためのAIプラットフォームです。
「チームの課題を可視化し、対話を生む」ことをコアバリューとしています。

## 2. コア機能
システムは大きく2つのフェーズで構成されています。

### Phase 1: ワークショップ (Workshop)
人間によるファシリテーションとAIによるリアルタイム分析を組み合わせ、チームの関係性を可視化します。
- **リアルタイム参加 (QR Code)**: 参加者はQRコードを読み込んでセッションに参加し、スマートフォンからリアルタイムに問いへ回答します。
- **構造化アンケート (AsIs/ToBe/Solution)**: 以下の3つの観点で回答を収集し、思考を整理・深掘りします。
  1. **As Is (現状)**: 現在のチームの状態や課題。
  2. **To Be (理想)**: 本来あるべき理想の姿。
  3. **Solution (解決策)**: ギャップを埋めるための具体的なアクション。
- **AI合意形成分析機能**: 収集した回答をリアルタイムで分析し、以下のインサイトを提供します。
  - **合意点 (Consensus)**: チームメンバー間で共通している認識。
  - **相違点 (Conflict/Gap)**: 認識が食い違っているポイント。
  - **ディスカッションポイント**: 優先的に対話を行うべきテーマの抽出。

### Phase 2: AIモニタリング (AI Monitoring)
ワークショップ後の状態を継続的に追跡・改善します。
- **週次パルスチェック**: 定期的な簡易アンケートでチームの状態を把握。
- **AI感情分析**: テキストデータから感情のトレンドを予測・分析。
- **インサイト提供**: マネージャー向けに具体的な改善アクションを提案。

## 3. 技術スタック
最新のモダンWeb技術を採用しています。
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **UI Library**: React 19, Tailwind CSS 4, Shadcn/ui
- **Icons**: Lucide React
- **AI**: Vercel AI SDK (OpenAI/Anthropic compatible)
- **Visualization**: Recharts

## 4. プロジェクト構造 (`app/`)
- `/page.tsx`: ランディングページ。製品の価値提案と機能紹介。
- `/login`: ログイン・認証画面。
- `/dashboard`: メイン機能へのアクセスポイント。
  - `/analytics`: 分析結果の表示。
  - `/monitoring`: 継続的なチーム状態のモニタリング。
  - `/workshops`: ワークショップの管理・実施。
  - `/surveys`: アンケートの作成・管理。
  - `/team`: チームメンバー管理。
- `/api`: バックエンドロジック (Serverless Functions)
  - `/workshop`: ワークショップ関連のAPI群
    - `/session`: セッション管理
    - `/generate-questions`: AIによる質問生成
    - `/analyze-responses`: AIによる回答分析
    - `/join`, `/question`, `/response`: 参加者インタラクション

## 5. デザイン・UX
- **コンセプト**: "Rich Aesthetics", "Premium", "Dynamic"
- **カラーパレット**: Teal & Lime gradient を基調としたモダンな配色。
